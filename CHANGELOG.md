# MixAgent Changelog

## v0.1.2
- 新增桌面端自动更新检查：应用启动后自动检测 GitHub Releases 上的新版本。
- 新增更新提示条与下载进度：发现版本后可直接下载，下载时实时显示百分比，完成后支持一键重启安装。
- 模型页收回为两栏工作区：左侧模型列表，右侧直接编辑配置，共享 API Key 逻辑继续保留在底层。

## v0.1.1
- `智能助手` 改为唯一可见入口，内部复用系统执行层。
- 默认记录员模板移除，改为系统级双轨留痕与成果索引。
- 新增“本次命令产物模式”：快速文本、单文件、多文件、工程文件。
- 群聊切换改为静默，不再写入“已导入项目”提示。
- 群聊会自动维护隐藏元数据、运行记录和成果索引。

## v0.1.0
- 首个桌面版可运行基线。
