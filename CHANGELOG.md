# MixAgent Changelog

## v0.1.6
- 工作流面板颜色收回主界面的深灰体系，入口按钮与展开面板风格统一。
- 左侧只保留简单/高级两种主模式说明，右侧按当前模式显示对应配置，不再一次性铺满全部策略。
- “编辑群成员”现在可直接进入当前群聊成员的智能体编辑页，减少中间跳转。
## v0.1.5
- 工作流配置区重做为内联双栏面板：左侧简介，右侧表格式配置，替代原来的小下拉菜单。
- 高级发言策略从设置中心迁回主工作区：轮次、顺序、上下文和摘要策略现在都在同一处直接编辑。
- 底部入口改为“讨论配置”摘要按钮，当前模式和本次产物一眼可见。
