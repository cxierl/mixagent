# MixAgent v0.1.2 Release Notes

## Highlights

### 1. Desktop auto update

Starting with this release, MixAgent supports a desktop auto-update flow:

- automatic check on startup
- throttled re-check when the window regains focus
- visible update prompt when a new version is found
- progress bar during download
- one-click restart and install after download

### 2. Leaner model page

The model page was reduced from three columns back to two:

- left: model list
- right: model configuration

The shared Base URL and API Key logic still exists internally, but it is no longer exposed as a dedicated provider column.

### 3. Existing configuration is preserved

This release keeps the existing rule that updates should not wipe user state. Configured:

- models
- API keys
- skills
- agents
- teams
- rooms
- workflow strategies

are expected to remain in local storage across updates.

## Design Intent

MixAgent is not a generic chat shell. It is a workspace for operating a small AI team through structured multi-role collaboration.

Core principles:

- one visible assistant
- one hidden system control layer
- persistent rooms
- persistent teams
- per-command output modes
- system-level trace and indexing instead of a recorder agent seat

## Installation

Double-click the installer for one-click setup.

## Update Behavior

If a user already has an older version installed and this release is uploaded to GitHub Releases, the app will detect the new version on the next startup.
