# MixAgent v0.1.2 发布说明

## 本次版本重点

### 1. 桌面端自动更新

从本版本开始，MixAgent 支持桌面端自动更新链路：

- 启动自动检查
- 窗口重新激活时按节流策略复查
- 检测到新版本后弹出更新提示
- 下载时显示进度条
- 下载完成后支持一键重启安装

### 2. 模型页收口

模型页从三栏收回为两栏：

- 左侧：模型列表
- 右侧：模型配置

共享 Base URL 和 API Key 的底层逻辑仍然保留，但不再暴露为独立服务商栏，减少使用成本。

### 3. 继续保留已有配置

本版本继续遵循“更新不冲掉用户配置”的原则，已配置的：

- 模型
- API Key
- 技能
- 智能体
- 团队
- 群聊
- 发言策略

默认都会保留在本地存储中。

## 设计意图

MixAgent 不是通用聊天壳，而是一个让用户驱动多角色 AI 团队协作的工作台。

核心原则：

- 一个可见助手
- 一个隐藏系统控制层
- 房间长期复用
- 团队长期复用
- 产物模式按次选择
- 系统级留痕代替记录员席位

## 发布资产

本版本发布时应上传：

- `MixAgent Setup 0.1.2.exe`
- `MixAgent Setup 0.1.2.exe.blockmap`
- `latest.yml`

## 安装方式

双击安装包即可完成一键安装。

## 更新方式

如果用户已经安装旧版本，且本版本已上传到 GitHub Release，应用下次启动后会自动检测到更新。
