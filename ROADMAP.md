# MixAgent Roadmap

## Next UX

1. Add optional light theme and theme presets.
2. Support panel docking modes (left/right/floating) for assistant.
3. Add keyboard command palette for common actions.

## Reliability

1. Persist all app state in project snapshots (agents/models/files/pipeline).
2. Add import/export for full workspace JSON.
3. Add model health checks with retry/backoff strategy.

## Collaboration

1. Add role templates for common teams (PM, Architect, Reviewer, Writer).
2. Add reusable skill packs per scenario.
3. Add discussion transcript search and filtering.

## Engineering

1. Split single-file app into `index.html` + `styles.css` + `app.js`.
2. Add basic test harness for parsing and action execution.
3. Add CI checks before GitHub release.
